Spacing & Grids

Consistent spacing and structure are essential to maintaining a clean, readable, and well-balanced layout across the Daisy Studios website. This system ensures design clarity and visual harmony, no matter the content or screen size.


The website follows a 12-column grid layout as the foundation for all sections and components. Designers and developers should align all elements to this grid to maintain alignment and modularity throughout the site.

All page margins are set to 40px on both sides. This margin ensures that content has breathing room and does not touch the edges of the screen, especially on smaller devices.

To create clear separation and rhythm between sections, a vertical break of 280px should be maintained. This spacing helps visually distinguish content blocks and allows for a smoother reading experience.

Media Formatting

All media assets on the Daisy Studios website including images and videos follow a consistent formatting system to maintain visual harmony and scalability across all screen sizes.


Every media file is formatted using a 16:9 aspect ratio, ensuring that visuals display predictably and cleanly in both desktop and mobile environments.

There are four standardized media sizes used throughout the site, ranging from large hero visuals to small content blocks.

By adhering to these formatting rules, Daisy Studios ensures media is both brand-consistent and highly adaptable across all platforms and devices.
